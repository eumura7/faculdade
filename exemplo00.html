HTML Linguagem de Marcação de Hipertexto

Tutoriais para iniciantes

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o signifcado e a estrutura do conteúdo da web. Outras tecnologias além do HTML geralmente são usadas para descrever a aparência/apresentação (CSS) ou a funcionalidade/comportamento (JavaScript) de uma página da web.

Tag Utilidade
